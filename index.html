Get Started Wiki

4 Steps

Step 1 - Setup Workplace

Make sure you have both Git and Nodejs installed.

Step 2 - Clone Repository

Run this command in your terminal:

git clone -b master https://github.com/andrejarrell/Starterbot.git

Step 3 - Setup Bot

Enter bot folder and run these commands:

npm install

node setup.js

Step 4 - Start Coding :)

Refer to the Wiki if you have questions!
